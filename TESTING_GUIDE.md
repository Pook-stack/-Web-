## 🔧 数据库表结构

### reviews 表
```sql
id (主键)
club_id (俱乐部ID，外键)
user_id (用户ID)
rating (评分，1-5)
comment (评价内容)
created_at (创建时间)
updated_at (更新时间)
UNIQUE(club_id, user_id) -- 防止重复评价
```

### game_categories 表
```sql
id (主键)
key (分类键值，唯一)
name (显示名称)
icon (图标)
sort_order (排序)
is_active (是否启用)
created_at (创建时间)
updated_at (更新时间)
```
